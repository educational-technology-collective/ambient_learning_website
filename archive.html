How to Start Acquiring Learnings Moments

1. Navigate to the Chrome Extension to Install the Ambient Learning Agent extension

2. Click "Add to Chrome"
Click "Add extension"

3. Click the extensions icon
Pin the extension for ease of access
Click on the Ambient Learning Agent extension

4. In the next steps ensure that you log in with your Umich google email
Click "Log In"
Click "Continue with Google"
Click "uniqname@umich.edu"

*** MAC USERS ***

There can sometimes be an issue where the popup closes before you log in so you can right click to inspect the popup. This will open another developer tools popup. At this point you should be able to log in with no issues. If you still have issues logging in to the Ambient Learning Agent extension, you can contact ambientlearning@umich.edu

5. Navigate to the SIADS 505 course page in Coursera

6. Start watching videos to gain learning moments
